Сегодня я научился входить в гит по SSH ключу
Для начала нужно сгенерировать ключь с помощью $ssh-keygen и согласиться со всеми требованиями с помощью Enter
Дальше нам покажут где лежит публичный "ключ название_ключа.pub"
Мы его читаем с помощью $cat, а дальше копируем содержимое публичного ключа в GitHub
А именно: Settings --> SSH and GPG keys --> New SSH key, и вставляем содержимое публичного ключа туда
Зетем на странице нашего репозитория выбираем Code --> SSH и копируем содержимое
После в терминале мы связываем локальный репозиторий и удаленный с помощью команды:
$git remote set-url origin git@github.com:keygen12/learning.git - (сюда вставляем то что мы скопировали из страницы прошлым действием)

А так же сегодня я научился объдинять ветки, это оказалось просто и без всяких ошибок
Первым делом мы должны переключиться на ту ветку к которой мы привязываем вторую ветку с помощью $git checkout название_ветки
Далее сама команда объединения ветки $git merge название_второй_ветки 
Остается лишь пропушить наше дело $git push. Коммитить наше изменение не нужно, но если хочется то $git add . $git commit -m "бла-бла-бла" $git push
