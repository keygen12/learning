#Написал программу по примерам из интернета которая при полном её разборе, учит навыкам поиска файлов, их сортировки, форматированию и работы с ними

```
find /home/eugene/ -type f -size +100M -printf '%20s %p\n' 2>/dev/null | sort -n | awk '{printf "%10.1f MB %s\n", $1/1024/1024, $2}'
```
Разберем последовательно:
1. Ищет в /home/eugene/ по типу(-type) файл (f), а так же по размеру (-size) который больше 100 мегабайт (+100M)
2. Описывает что и как выводить (-printf). А именно размер выровненный до 20 символов (%20s), а так же  путь к файу (%p)
3. Если появляются ошибки в stderr по типу "Отказано в доступе", то перенаправлять (2>) эти ошибки в /dev/null, это своего рода корзина
4. Сортируем по числу (по размеру) с помощью ```
	sort -n```
5. Производим форматирование текста по ширине 10 (%10) с одним знаком числа после запятой (.1f). Далее мы говорим куда подставить строку с помощью (%s), и в конце осуществляем перенос строки на новую с помощью (\n)
6. Значение первого столбца ($1) мы делим на 1024 два раза, чтобы результат выводился в Мегабайтах. И в конце выводим второй столбец с расположением файла
